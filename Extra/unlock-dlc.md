# Unlock DLC

## Quick Method

Try dragging and dropping the DLC AppID or DLC link onto SteamTools, just like dragging and dropping a file.

Example:

- `https://steamdb.info/app/3174120/` -> drag into SteamTools

## How To Check If It Can Be Added

Use [SteamDB](https://steamdb.info/app/1144200/) and check the **Depots** tab.

If a DLC AppID has **no depot**, you can usually add it.

### Step-by-step

1. Open the game page on SteamDB.
2. Go to the **DLCs** section and open one DLC page.
3. Copy the DLC AppID (the number in the URL).
4. Go back to the main game page and open **Depots**.
5. Search for that DLC AppID in the depots list.
6. If there is no depot for that AppID, you can add it.

## Notes

- The top two common redistributables like **VC 2019 Redist** and **DirectX Jun 2010 Redist** do not need to be added.
- Always verify the exact AppID before dragging it into SteamTools.
